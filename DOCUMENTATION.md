# سافٹ ویئر فلو اور دستاویزات

## سسٹم کا جائزہ

یوسف اینڈ کو بزنس مینجمنٹ سسٹم ایک مکمل بزنس مینجمنٹ سولیوشن ہے جو چھوٹے اور درمیانے کاروباروں کے لیے ڈیزائن کیا گیا ہے۔

## سافٹ ویئر فلو

### 1. صارف لاگ ان
```
لاگ ان پیج → یوزرنیم/پاس ورڈ → تصدیق → ڈیش بورڈ
```

### 2. کھاتے کا انتظام
```
نیا کھاتہ → معلومات درج کریں → محفوظ کریں → کھاتہ بن گیا
کسٹمر لسٹ → تلاش/فلٹر → تفصیلات دیکھیں/ایڈٹ کریں
```

### 3. جنس کا انتظام
```
نیا ائٹم → تفصیلات درج کریں → محفوظ کریں → سٹاک میں شامل
جنس لسٹ → سٹاک دیکھیں → ایڈٹ/حذف
```

### 4. خرید کا عمل
```
نیا خرید → سپلائر منتخب کریں → جنس شامل کریں → قیمت/مقدار → محفوظ کریں
→ سٹاک میں اضافہ → اکاؤنٹ میں ڈیبٹ
```

### 5. فروخت کا عمل
```
نیا سیل → کسٹمر منتخب کریں → جنس شامل کریں → قیمت/مقدار → محفوظ کریں
→ سٹاک میں کمی → اکاؤنٹ میں کریڈٹ
```

### 6. لین دین
```
کیش بنام → اکاؤنٹ منتخب کریں → رقم → محفوظ
کیش جمع → اکاؤنٹ منتخب کریں → رقم → محفوظ
جرنل واؤچر → دونوں اکاؤنٹس → رقم → محفوظ
```

### 7. رپورٹس
```
پارٹی لیجر → اکاؤنٹ منتخب → تاریخ رینج → رپورٹ
سٹاک کھاتہ → جنس منتخب → تفصیلات
بیلنس شیٹ → تاریخ → بیلنس
کیش بک → تاریخ رینج → تمام لین دین
```

## ڈیٹا بیس اسٹرکچر

### Users Table
- صارفین کی معلومات
- لاگ ان کی تفصیلات
- صارف کی قسم

### Accounts Table
- کھاتوں کی معلومات
- رابطہ کی تفصیلات
- افتتاحی بیلنس

### Items Table
- جنس کی معلومات
- قیمتیں
- سٹاک کی معلومات

### Purchases Table
- خرید کی تفصیلات
- سپلائر کی معلومات
- کل رقم

### Purchase Items Table
- خرید کی لائن آئٹمز
- جنس، مقدار، قیمت

### Sales Table
- فروخت کی تفصیلات
- کسٹمر کی معلومات
- کل رقم

### Sale Items Table
- فروخت کی لائن آئٹمز
- جنس، مقدار، قیمت

### Transactions Table
- تمام لین دین
- قسم (ڈیبٹ/کریڈٹ/جرنل)
- حوالہ

### Stock Movements Table
- سٹاک کی تمام حرکات
- خرید/فروخت/ایڈجسٹمنٹ
- موجودہ بیلنس

## فیچرز کی تفصیل

### ڈیش بورڈ
- آج کی فروخت/خرید کا خلاصہ
- کل کھاتے/جنس
- کم سٹاک کی وارننگ
- فوری لنکس

### کھاتے
- کسٹمر/سپلائر کا انتظام
- یوزر ٹائپس
- رابطہ کی معلومات
- افتتاحی بیلنس

### جنس
- جنس کی تفصیلات
- خرید/فروخت کی قیمتیں
- سٹاک کا انتظام
- کم از کم سٹاک

### خرید
- خرید کا ریکارڈ
- متعدد جنس شامل کرنا
- ڈسکاؤنٹ
- ادائیگی/بیلنس

### فروخت
- فروخت کا ریکارڈ
- متعدد جنس شامل کرنا
- ڈسکاؤنٹ
- وصولی/بیلنس

### لین دین
- کیش بنام (Debit)
- کیش جمع (Credit)
- جرنل واؤچر (JV)
- تمام لین دین کی فہرست

### رپورٹس
- پارٹی لیجر
- سٹاک کھاتہ
- سٹاک لیجر
- بیلنس شیٹ
- کیش بک
- روزنامچہ

## سیکورٹی

- SQL Injection Protection (PDO)

### پہلی بار استعمال

5. کھاتے بنائیں

4. رپورٹس چیک کریں
5. سٹاک چیک کریں
